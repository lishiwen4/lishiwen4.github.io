###1. 5G频段

5G是指的另外一个可以被免费使用的无线电频段（5150MHz - 5825MHz）

wifi 的5G 频段共划分为4个band

1. band1 ： 5170MHz - 5250MHz 对应的channel为
   + ch34		5170MHz
   + ch36		5180MHz
   + ch38		5190MHz
   + ch40		5200MHz
   + ch42		5210MHz
   + ch44		5220MHz
   + ch46		5230MHz
   + ch48   	5240MHz
2. band2 ： 5250MHz - 5330MHz
   + ch52		5260MHz
   + ch56		5280MHz
   + ch60		5300MHz
   + ch64		5320MHz
3. band3 ： 5490MHz - 5730MHz
   + ch100	5500MHz
   + ch104	5520MHZ
   + ch108	5540MHz
   + ch112	5560MHz
   + ch116	5580MHz
   + ch120	5600MHz
   + ch124	5620MHz
   + ch128	5640MHz
   + ch132	5660MHz
   + ch136	5680MHz
   + ch140	5700MHz
   + ch144	5720MHz
4. band4 ： 5735MHz - 5835MHz
   + ch149	5745MHZ
   + ch153	5765MHz
   + ch157	5785MHz
   + ch161	5805MHz
   + ch165	5825MHz
